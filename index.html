Tetris

A multiplayer tetris game written in Python for Linux and Mac OS

Download Download fork Tetris on GitHub

Install Instructions

First make sure that you have python 2.7. If you do not you can obtain it here: http://www.python.org/download/ Next follow these instructions:

$ cd <the directory where you want tetris>
$ git clone git://github.com/fen11/Tetris.git

Use Instructions

$ cd <the directory where you installed tetris>/Tetris
$ ./tetris

If you want to play alone or be a game host, run both the server and the client and give the room name to anyone else who is playing. Next click ok to run the server, then click ok to start the client. If you want to run only the client, and connect to someone else's server, select only client and then enter the room name that the host gives you and click ok.
